# Roadmap

Roadmap фиксирует границы этапов, а не календарные обещания. Перенос функции между этапами требует обновления этого файла и, при изменении поведения, продуктового решения.

## Текущий milestone — устойчивый mock-сценарий

- рабочий адаптивный mock-интерфейс;
- исправление найденных UX-багов;
- mock CRUD проектов с серверной проверкой роли;
- mock-выбор файлов и безопасные метаданные;
- доменная модель статусов;
- фильтр истории по проекту;
- безопасный ручной повтор после timeout с новым idempotency key;
- regression-тесты для каждого исправленного QA-наблюдения;
- актуальная документация как источник контекста между задачами.

## Следующий milestone — Supabase

- Supabase Auth;
- постоянное хранение;
- RLS;
- production-роли администратора и редактора;
- Supabase-реализации репозиториев;
- миграции и проверка матрицы доступа.

## Следующий интеграционный milestone — Bitrix24

- настоящий webhook Bitrix24;
- получение пользователей и рабочих групп;
- создание задачи;
- реальная загрузка файлов;
- получение актуального статуса;
- синхронизация истории;
- contract tests на очищенных ответах портала.

## Не входит сейчас

- аналитика;
- клиентский кабинет;
- velocity;
- план-факт;
- AI;
- несколько порталов;
- OAuth;
- offline;
- push-уведомления.
